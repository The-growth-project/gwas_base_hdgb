# Base GWAS
Genome-wide analysis base pipeline for the [Norwegian Mother, Father and Child Cohort Study (MoBa)](https://www.fhi.no/en/ch/studies/moba).
The documentation corresponds to the analyses version `2026.02.05`.
### Birth weight test
GWAS of birth weight

- [children](birth_weight/pop_children_pheno_weight_birth.md): GWAS of Birth weight against the genome of children.
- [mothers](birth_weight/pop_mothers_pheno_weight_birth.md): GWAS of Birth weight against the genome of mothers.
- [fathers](birth_weight/pop_fathers_pheno_weight_birth.md): GWAS of Birth weight against the genome of fathers.
- [parents](birth_weight/pop_parents_pheno_weight_birth.md): GWAS of Birth weight against the genome of parents.
### Birth weight test PCs
